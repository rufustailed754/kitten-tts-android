## 🧰 What this app does

KittenTTS is an Android text-to-speech app that runs on your device. It turns written text into speech without sending your text to a server.

It is built for people who want:

- Local text-to-speech on Android
- Fast voice output
- Simple speech playback
- A private speech tool that works offline after setup

## ✅ What you need

Before you install the app, check these basics:

- An Android phone or tablet

## 🎙️ Using the speech tool

KittenTTS is made for simple text input and voice output.

Typical use:

1. Paste text from notes, articles, or messages.
2. Split long text into smaller parts if needed.
3. Choose the voice that sounds best to you.
4. Press play to hear the text read aloud.
5. Stop playback when you do not need more audio.

The app uses on-device speech processing, so it does not need to send your text to an online service for playback.

## 🧪 Basic troubleshooting

If the app does not work as expected, try these checks:

- No sound: raise media volume and check silent mode
- App will not install: confirm that Android allows app installs from the source you used
- Speech is slow: close other apps and try shorter text
- Voice does not load: check that all app files finished downloading
- App closes right away: restart the phone and open the app again

If audio sounds too fast or too slow, change the speech settings and test again with a short sentence.

## 🧩 How the app works

The app takes text you enter, sends it through local speech logic, and plays the voice on your device.

Core parts:

- Kotlin handles the app code
- Jetpack Compose builds the screen
- ONNX Runtime runs model-based speech work
- espeak-ng provides speech synthesis support

This setup keeps the speech flow inside the device instead of relying on a remote service
